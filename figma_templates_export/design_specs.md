# AI智汇观察 - 设计规范文档

## 🎨 设计系统

### 1. 色彩系统
```css
/* 主色 */
--primary-blue: #2962ff;    /* 科技蓝 - 标题、重点 */
--dark-blue: #1a237e;       /* 深蓝 - 背景、副标题 */
--accent-orange: #ff6d00;   /* 亮橙 - 强调、CTA */

/* 中性色 */
--light-gray: #f5f5f5;      /* 浅灰 - 背景、卡片 */
--medium-gray: #9e9e9e;     /* 中灰 - 次要文字 */
--dark-gray: #424242;       /* 深灰 - 正文文字 */
--white: #ffffff;           /* 白色 - 文字、背景 */

/* 语义色 */
--success: #4caf50;         /* 成功 - 增长、正面 */
--warning: #ff9800;         /* 警告 - 注意、风险 */
--error: #f44336;           /* 错误 - 下降、负面 */
--info: #2196f3;            /* 信息 - 中性信息 */
```

### 2. 字体系统
```
标题层级:
  H1: 32px / 1.2 / 700     # 主标题
  H2: 24px / 1.3 / 600     # 二级标题
  H3: 20px / 1.4 / 600     # 三级标题
  H4: 18px / 1.4 / 500     # 小标题

正文层级:
  Body Large: 16px / 1.6 / 400   # 正文大
  Body: 14px / 1.6 / 400         # 正文
  Body Small: 12px / 1.5 / 400   # 正文小

标注层级:
  Caption: 12px / 1.4 / 400      # 标注
  Footnote: 11px / 1.4 / 400     # 脚注
```

### 3. 间距系统 (8px网格)
```
大间距: 32px   # 区块之间
中间距: 16px   # 元素之间
小间距: 8px    # 紧密元素
微小间距: 4px  # 图标与文字
```

## 🎯 设计最佳实践

### 1. 可访问性
- 色彩对比度 > 4.5:1
- 字体大小 > 12px
- 清晰的视觉层次
- 替代文本描述

### 2. 响应式考虑
- 重要内容在首屏显示
- 移动端友好的触摸目标 (>44px)
- 避免过小的文字和元素
- 考虑不同设备的显示效果

### 3. 性能优化
- 图片压缩和优化
- 减少不必要的装饰元素
- 使用矢量图形代替位图
- 考虑加载速度和文件大小

### 4. 品牌一致性
- 统一使用品牌色彩
- 一致的字体和间距
- 可识别的视觉语言
- 符合品牌调性的语气
