# 工作安排管理系统

## 记录格式说明

### 长期项目记录格式
```
# [项目名称]
**目标：** [项目总体目标]
**时间范围：** [开始日期] - [预计完成日期]

## 阶段划分
### 阶段一：[阶段名称]
- **时间：** [时间段]
- **目标：** [本阶段目标]
- **关键任务：**
  1. [任务1]
  2. [任务2]
  3. [任务3]
- **完成标准：** [验收标准]

### 阶段二：[阶段名称]
...

## 每日/每周任务
- [日期]：[具体任务]
- [日期]：[具体任务]

## 进展跟踪
- [日期]：[进展更新]
- [日期]：[遇到的问题/解决方案]
```

### 日常任务记录格式
```
## [日期] 工作安排
### 上午
- [ ] [任务1]
- [ ] [任务2]

### 下午  
- [ ] [任务1]
- [ ] [任务2]

### 长期项目跟进
- [项目名称]：[今日需要完成的部分]
```

## 提醒系统
- 每天12:30：提醒上午工作完成情况和下午计划
- 每天19:00：提醒全天工作完成情况和明日计划
